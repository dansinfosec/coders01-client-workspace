# seo-audit

**Stage 3** tool. Analyzes scraped pages (or the live site) and writes audit
reports to `clients/<client>/analysis/`.

Checks:
- **SEO** — titles, meta descriptions, headings, canonicals, schema, sitemap.
- **Performance** — page weight, render-blocking resources, image sizing.
- **Accessibility** — landmarks, alt text, contrast, keyboard nav, ARIA.
- **Content** — inventory, content types, duplication, gaps.
- **Tech** — stack, CMS, third-party scripts, forms, integrations.
