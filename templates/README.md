# templates/

Reusable starters shared across all clients. Copy from here into a client's
`rebuild/` folder — never edit a client's copy back into these templates unless
the change is genuinely reusable.

- **django-react-starter/** — base Django (backend/API) + React (frontend)
  project skeleton. The starting point for every rebuild.
- **tailwind-components/** — reusable Tailwind CSS UI components (buttons, cards,
  nav, forms, hero sections, etc.).
- **seo/** — SEO boilerplate: meta tags, Open Graph / Twitter cards, JSON-LD
  schema, `robots.txt`, and sitemap scaffolding.
