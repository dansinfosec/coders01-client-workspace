# ai-content-generator

**Stage 4** tool. AI-assisted content generation, using prompts from `prompts/`.

Use cases:
- Rewrite / improve page copy for clarity and conversion.
- Generate meta titles + descriptions at scale.
- Generate image `alt` text from context.
- Draft the content model and rebuild spec from the analysis.

Writes to `clients/<client>/docs/` (specs) and can propose copy for the rebuild.
